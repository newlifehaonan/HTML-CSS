head

  • You can define `Style`,`meta`,`title`,`base`,`link`,`script` in head
  • meta
    • define basic information such as `author`,`description`,`generator` etc. for search
    • has two main attribute `name` and `content`
    • relocate web page use `http-equiv = refresh` and `url = relocate URL`
  • base
    • use `base` to define a default linked page
  • script
    • use `script` to insert outer resource in html

body
